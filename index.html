// Live Response
K
ask "explain UVM_ACTIVE vs UVM_PASSIVE agent"
AI
UVM_ACTIVE: driver + sequencer + monitor. Drives DUT stimulus.

UVM_PASSIVE: monitor only. Observes, no driving.

Set via: agent.is_active = UVM_PASSIVE in env build_phase.
K
ask "how to connect virtual interface in UVM"
AI
Set in top: uvm_config_db#(virtual axi_if)::set(null, "uvm_test_top.*", "vif", inst);

Get in component: uvm_config_db#(virtual axi_if)::get(this, "", "vif", vif);

Always call get() in build_phase.
