// Live Response
K
ask "explain the difference between UVM_ACTIVE and UVM_PASSIVE agent"
AI
UVM_ACTIVE agent instantiates driver + sequencer + monitor — it actively drives DUT stimulus.

UVM_PASSIVE agent only has a monitor — it observes signals without driving.

Use PASSIVE on DUT outputs you want to sample without interference.
K
ask "how do I connect virtual interface in UVM"
AI
Use uvm_config_db.

Set: uvm_config_db#(virtual axi_if)::set(null, "uvm_test_top.*", "vif", axi_if_inst);

Get: uvm_config_db#(virtual axi_if)::get(this, "", "vif", vif);

Always call get() in build_phase.
